# **App Name**: Report Master

## Core Features:

- Report Generation: Generate daily, weekly, and custom date range reports based on structured data inputs. This tool will format the content according to the example provided in the prompt.
- Data Input Forms: Create user-friendly forms to input daily activity reports, challenge logs, and recommendation data from various units.
- Report Preview: Display a preview of the generated report before final submission to ensure accuracy and completeness.
- Customizable Report Templates: Enable users to select from or create report templates to suit different reporting needs. This involves options to adjust sections such as security situation, activities, recommendations, and challenges.
- Date-Specific Reporting: Allow users to filter and generate reports for specific days, weeks, or custom date ranges.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke a sense of authority and professionalism, nodding to the app's purpose of creating official reports.
- Background color: Light gray (#F5F5F5), a muted backdrop that ensures comfortable readability and places emphasis on the content itself.
- Accent color: Vivid orange (#FF5722) to highlight critical information and key action items, commanding attention without overwhelming the professional tone.
- Font pairing: 'Inter' (sans-serif) for both headlines and body text to maintain clarity, neutrality, and modern appeal across all content.
- Clean and structured layout with distinct sections for each reporting unit (e.g., Gasabo DPU, Kicukiro DPU) to ensure organized presentation.
- Use minimalist icons to represent different report sections or data categories, enhancing user navigation and overall comprehension.
